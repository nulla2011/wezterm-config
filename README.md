## 前提条件

安装`wezterm`终端

[WezTerm终端](https://github.com/wez/wezterm/releases)

安装目录：C:\soft\WezTerm-windows



## 使用方法

1、下载本储存库压缩包

2、解压压缩包

3、将解压的文件放入：`$HOME/.config/wezterm`目录底下

Windows目录：C:\Users\Fizz\\.config\wezterm
